Sorting Algorithm Visualizer - Instructions

What is this?
This is a Python-based sorting visualizer that uses Pygame to animate sorting algorithms in real time. It currently supports Bubble Sort, with more algorithms coming soon.

How to Use:
1. Run the script using Python.
2. Press the SPACE key to start sorting.
3. Close the window at any time to exit.

Before Running:
- Ensure you have Python installed.
- Install dependencies using: pip install pygame
- Run the script: python sorting_visualizer.py
